# El último incendio

En la parte de "quizá pueda despegar...":

- Para el acorde Am, jugar a levantar el dedo 3.
- Para el acorde D, jugar a poner el dedo 4 en traste 3 de primera cuerda.

**Intro**
```
G Cadd9 G
```
